# 寻色 · 中国传统色工具站

一色一诗 — 中国传统色彩浏览器。既是设计灵感工具，也是可盈利的产品。

## 技术栈

- 纯原生 HTML/CSS/JS，零依赖，单文件
- 直接双击 `index.html` 打开
- Canvas 2D 金色粒子背景
- 丝绸纹理 CSS（色卡展示）
- 配色方案收集 + Canvas 导出 PNG

## 产品功能

1. **色彩浏览** — 按色系分类（赤橙黄绿青蓝紫褐黑白灰金），11 类 ~80+ 色
2. **搜索** — 按名字搜索颜色
3. **详情弹窗** — 丝绸纹理色卡、古诗出处、一键复制 HEX/RGB
4. **配色方案** — 最多选 5 色，底部工具栏收集，一键导出 PNG
5. **盈利入口** — 赞赏 ¥9.9、开发者 API 预告
